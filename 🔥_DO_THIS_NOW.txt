🚀 DO THIS NOW (5 MINUTES):

STEP 1: Open MongoDB Atlas
═══════════════════════════════════════════════════════════════════════════════
Go to: https://www.mongodb.com/cloud/atlas/register

Or click here: https://account.mongodb.com/account/register


STEP 2: Sign Up (Choose one)
═══════════════════════════════════════════════════════════════════════════════
• Sign up with Google (fastest!)
• Sign up with GitHub
• Sign up with email


STEP 3: Create Free Cluster
═══════════════════════════════════════════════════════════════════════════════
After login:
1. Click "Build a Database"
2. Choose "M0 FREE" (Shared)
3. Select AWS or Google Cloud
4. Choose region closest to you
5. Click "Create"


STEP 4: Create Database User
═══════════════════════════════════════════════════════════════════════════════
You'll see "Security Quickstart":
1. Username: bookadmin
2. Password: Click "Autogenerate Secure Password"
3. COPY THE PASSWORD! (you'll need it)
4. Click "Create User"


STEP 5: Set Network Access
═══════════════════════════════════════════════════════════════════════════════
1. Click "Add My Current IP Address"
2. Also click "Add IP Address"
3. Enter: 0.0.0.0/0
4. Description: Allow all
5. Click "Add Entry"
6. Click "Finish and Close"


STEP 6: Get Connection String
═══════════════════════════════════════════════════════════════════════════════
1. Click "Connect" on your cluster
2. Choose "Connect your application"
3. Copy the connection string
4. It looks like:
   mongodb+srv://bookadmin:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority


STEP 7: Modify Connection String
═══════════════════════════════════════════════════════════════════════════════
Change this:
mongodb+srv://bookadmin:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority

To this (add /bookmanagement before the ?):
mongodb+srv://bookadmin:YOUR_ACTUAL_PASSWORD@cluster0.xxxxx.mongodb.net/bookmanagement?retryWrites=true&w=majority

Replace:
• <password> with your actual password
• Add /bookmanagement before the ?


STEP 8: Update Your .env File
═══════════════════════════════════════════════════════════════════════════════
1. Open: backend/.env
2. Find the line: MONGO_URL=...
3. Replace it with your connection string
4. Save the file


STEP 9: Restart Backend
═══════════════════════════════════════════════════════════════════════════════
The backend will automatically restart (nodemon is watching)
You should see: "Connected to MongoDB"


STEP 10: Test It!
═══════════════════════════════════════════════════════════════════════════════
1. Open: http://localhost:3000
2. Click "Register"
3. Create an account
4. Add a book!
